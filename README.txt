Tetris game
===========
This is a Tetris game developed using Python and pygame.
Operational requirements:
1. Install Python 3.6 or higher version
2. Install pygame library (pip install pygame)
Operation method:
1. Double click the "Start Tetris. bat" file
Game controls:
-Left and right directional keys: move blocks
-Up arrow key: Rotate the block
-Down arrow key: accelerate descent
Wishing you a pleasant gaming experience!
